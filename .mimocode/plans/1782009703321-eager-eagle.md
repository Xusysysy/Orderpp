# Plan: 菜品加号按钮 + 飞行动画 + 账单红圈 + 页切换过渡

## Modified Files

| File | Change |
|------|--------|
| `app/.../ui/component/MenuCard.kt` | 添加"+"按钮（客人模式），点击后切换为 QuantityStepper |

| `app/.../ui/screen/main/MainScreen.kt` | 页面过渡动画、飞行动画、账单红圈 Badge |

## Step 1: MenuCard 加号按钮

修改 `MenuCard`：
- 新增参数：`showAddButton: Boolean`, `orderQuantity: Int`, `onAddClick: () -> Unit`, `onIncrement: () -> Unit`, `onDecrement: () -> Unit`
- 卡片底部 Row：如果已点过（`orderQuantity > 0`）显示 QuantityStepper，否则显示一个小"+"圆形按钮（`FilledIconButton`）
- 卡片主体点击仍然打开配方
- 只有客人模式 `showAddButton = true` 才显示加号

## Step 2: RecipeSheet 加号按钮

修改 `RecipeSheet`：
- 新增参数：`isStaff: Boolean`, `orderQuantity: Int`, `onAddClick: () -> Unit`, `onIncrement: () -> Unit`, `onDecrement: () -> Unit`
- 在"步骤"区域下方添加加号/数量控制器（仅客人模式）
- 点击加号 → 调 `onAddClick` 并 `onDismiss`

## Step 3: 账单按钮红圈 Badge

修改 `MainScreen` 的 `NavigationBarItem` for BILL：
- 使用 `BadgedBox`（Material3）包裹 icon
- 当 `currentOrder?.items.isNotEmpty()` 时显示红圈 Badge
- Badge 内容为 `currentOrder?.items?.sumOf { it.quantity } ?: 0`

## Step 4: 飞行动画

在 `MainScreen` 中：
- 新增状态：`var flyTarget by remember { mutableStateOf<MenuItemEntity?>(null) }` 和 `var flySourceOffset by remember { mutableStateOf(Offset.Zero) }`
- 客人点击加号时：设 `flyTarget = item`, `flySourceOffset = sourceOffset`
- 如果当前在 RecipeSheet 中，先关闭 Sheet；如果不在菜单页，切换到 `Tab.MENU`
- 在 `Scaffold` 外层叠加一个 `Box`，当 `flyTarget != null` 时显示飞行动画：
  - 在 `flySourceOffset` 位置绘制一个小"+"图标/文字
  - 使用 `Animatable` 从 source offset 动画到账单 button 的 `onGloballyPositioned` 记录的绝对位置
  - 动画时长 ~400ms，带贝塞尔曲线效果（先上抛再落下）
  - 动画结束后清除 `flyTarget`

- 账单按钮位置获取：在 `NavigationBarItem(Bill)` 上使用 `onGloballyPositioned` 记录其全局坐标到状态

## Step 5: 页面切换过渡动画

修改 `MainScreen` 的 tab 内容区域：
- 使用 `AnimatedContent(targetState = selectedTab)` 包裹 `when(selectedTab)` 块
- 过渡效果：`slideInHorizontally { it }` + `fadeOut()` / `slideOutHorizontally { -it }` + `fadeIn()`
- 方向：右滑进、左滑出（菜单←→账单←→我的）

## Step 6: 菜单页面订单信息同步

`MenuTabContent` 已通过 `currentOrder` 和 `totalPrice` 状态响应式显示订单。需要：
- 在菜单网格上方添加一个订单摘要横幅（当前订单 N 件 / ¥total），仅当有订单时显示
- 点击横幅可切换到账单页

## Step 7: 员工屏蔽 + 号

所有加号按钮和飞行效果只在 `!isStaff` 时生效。`MenuCard` 和 `RecipeSheet` 通过参数控制。
